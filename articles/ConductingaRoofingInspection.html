The SkyCam - Conducting a Roofing Inspection

 

When conducting a roofing inspection there is a lot to remember, these are the key stages:

  • Ensure that the location is suitable for aerial work
  • Understand the type of inspection and key goals
  • Ensure that all key features are included
  • Work with other professionals as appropriate

The following features, if present, should be included:

  • Aerials
  • Chimney Pots
  • Chimneys
  • Fan Vents
  • Fascias
  • Felt
  • Fixings
  • Flashing
  • Flat Roofs
  • Flaunching
  • Guttering
  • Lightning Conductors
  • Parapets
  • Pitched Roofs
  • Ridge Tiles
  • Satellite Dishes
  • Sheeting
  • Slates
  • Solar Panels
  • Valleys
  • Windows
